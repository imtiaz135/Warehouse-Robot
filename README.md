# 🤖 Warehouse Robot using Q-Learning

## 📌 Project Overview

This project simulates a smart warehouse robot that learns how to pick and deliver items using **Q-learning (Reinforcement Learning)**.

The robot explores a grid environment, avoids obstacles, and gradually learns the most efficient path.

---

## 🎯 Key Features

* 🧠 Q-learning based intelligent decision making
* 🖱️ Interactive mouse-click selection
* 🔲 Obstacle-aware navigation
* 📊 Learning performance graph
* 🎮 Real-time robot simulation

---

## 🧩 How It Works

* The warehouse is a **grid environment**
* The robot learns using rewards & penalties
* It first explores randomly, then improves over time
* Finally, it finds the **optimal path**
